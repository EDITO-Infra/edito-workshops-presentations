# VSCode Service Example

This example shows how to use VSCode service in EDITO Datalab for mixed R/Python projects.

## üéØ What You'll Learn

- Launch VSCode service in EDITO Datalab
- Work with both R and Python in the same environment
- Use the integrated terminal and file explorer
- Run the provided examples

## üöÄ Getting Started

### Step 1: Launch VSCode Service
1. Go to [datalab.dive.edito.eu](https://datalab.dive.edito.eu/)
2. Select "VSCode" from the service catalog
3. Configure resources (CPU, memory) as needed
4. Launch the service

### Step 2: Open the Examples
1. Use the file explorer to navigate to the examples
2. Open the R scripts in `r_scripts/`
3. Open the Jupyter notebook in `jupyter_notebooks/`

### Step 3: Run the Examples
1. **R Scripts**: Use the integrated R terminal or run in RStudio
2. **Python/Jupyter**: Use the integrated Python terminal or Jupyter extension
3. **Mixed Projects**: Use both languages in the same workspace
